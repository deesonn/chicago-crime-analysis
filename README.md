# Chicago Crime Analysis

## Project Goal
Identify spatial and temporal crime hotspots in Chicago using reported crime data to uncover patterns by location, time, and offense type.

## Data Source
City of Chicago Open Data Portal (Crimes – 2001 to Present)

## Methods
- Data cleaning and preprocessing in Python
- Temporal feature engineering (year, month, hour, weekday)
- Spatial visualization using interactive maps
- Pre-COVID (2015–2019) vs Post-COVID (2020–2024) comparison

## Tools
- Python
- pandas
- folium / plotly
- Jupyter Notebook
